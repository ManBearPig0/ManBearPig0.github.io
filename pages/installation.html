Getting Started!

Composer

Composer is a Dependency management tool for PHP. You can use it to easily install and keep track of 'packages', like Laravel.
To install composer, first visit Download Composer and install the program on your computer. Either use the windows installer from here or install it manually.


Installing laravel with Composer

We can now use Composer to automatically install laravel for us. Open a console panel and go the the directory where you wish to install Laravel. This can be done using the cd command. Then execute these commands for creating your first Laravel project:

composer create-project laravel/laravel [project name]
cd [project name]

Now that your project has been created, you can use the serve command to start Laravel's local development server.

php artisan serve
